Gutzaria 2.8.5

We are ready to present you a new version of GutzFor - GutzFor 2.8.5! New features and improvements included in this version:

Changes:

  1. Changed the reading and handle of vpm! *
  2. Add output for local reduced one-body density matrix! *
  3. Add a new calculation mode for dChidR0 which needs a structure file! *
  4. Fixed a severe bug relating to dNFdR0! *
  5. Speed up dNFdR0!
  6. Add rotation matrix (like T, C3z, T*c2T) for nonequivalent atoms instead of independent bose calculation!
  7. Fixed one bugs relating to mpi and time reversal representation!
